# --> Features:
##      * Supported elements (using Roslyn parsing):
###         / Classes (and their variables)

###         / Structs (and their variables)

###         / Interfaces

###         / Enums (and their elements)

###         / Methods (and their parameters)


##      + XML documentation generation:
###         - Gives you a comfortable, easy and fast way to write and inject XML documentation into your scripts.
        
###         - Has auto-saving, backups, and batch file selection.
    

##      + Project structure file generation:
###         - Writes a txt file with your project's structure on it. This structure contains anything, like scripts, prefabs, assets, etc.

###         - This structure has a visually clear hirearchy.


##      + Project Information file generation:
###         - Writes a txt file with your project's information on it. This information is similar to the "Project structure", but it only contains the Supported elements. It also adds the information from your XML documentation for easy reading and comprehension.

###         - This structure has a visually clear hirearchy.


##      + Namespace generation:
###         - Inserts namespaces into all the scripts in a folder hirearchy (folder and subfolders).
